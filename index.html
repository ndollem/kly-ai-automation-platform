Standardisasi Data

Phase 1 — Isolasi Kredensial & Model Context Protocol

Keamanan data adalah prioritas utama. Melalui arsitektur **Model Context Protocol (MCP)** yang terstandar, asisten cerdas kita tidak pernah menyentuh database utama secara langsung, melainkan melalui jembatan konektor yang terenkripsi dan terisolasi.

  • ✔ Zero API Exposure: AI tidak memegang API Keys atau password database secara langsung.
  • ✔ Konektor Reusable: Sekali konektor GA4 dibangun, ia dapat digunakan kembali oleh agen atau asisten mana pun di masa mendatang.

Persyaratan Teknis & Infrastruktur Setup

Kami merancang setup arsitektur KLY Assistant agar sangat ringan, modular, dan dapat dijalankan di atas infrastruktur server standar yang sudah ada di kantor KLY saat ini.

Spesifikasi Infra Ringkas
🖥

Server Host VPS (Hemat)

Cukup Virtual Machine standar (1 vCPU, 1GB RAM) untuk menjalankan container Docker server asisten secara andal.

⚙

Software Stack

Python 3.10+, Node.js 18+, Docker Engine (standard containerized runtime).

🔒

Gateway & LLM API

Cukup 1 Nomor WhatsApp Biasa (menggunakan library open-source tanpa biaya API pihak ke-3 / Twilio / WABA) + Google Chat API + OpenAI / OpenRouter API Key.
